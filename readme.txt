Projeto: Trabalho Final de Linguagens de Marcação Extensíveis
Este projeto foi desenvolvido utilizando a IDE IntelliJ e testado no VSCode, especificamente no ambiente Windows.
Certifique-se de ter o JDK versão 18.0.2 instalado para executar o projeto no IntelliJ, e as dependências Java
necessárias configuradas no VSCode.

Arquivos Utilizados no Projeto:
AdicionarDOCTYPE.java
CombineXML.java
ValidaXMLcomDTD.java
ValidaXMLcomXSD.java
Consultas.java
CustomErrorHandler.java
TransformaXMLemHTML.java
Main.java

Arquivos Externos ao Projeto
Os seguintes arquivos externos foram utilizados no projeto e devem ser organizados conforme descrito abaixo:

Arquivos XML de Notas: nota1.xml, nota2.xml, ... Certifique-se de que todos os XML de notas estejam numerados
corretamente em ordem crescente e localizados dentro da pasta do projeto.
Arquivo DTD: nota_fiscal.dtd
Arquivo XSD: nota_fiscal.xsd
Arquivos XSL para Transformação em HTML: notas_ind.xsl e notas_tot.xsl

Passos para Instalação e Execução

Pré-requisitos
JDK versão 18.0.2 instalado
Dependências Java configuradas no VSCode

Instalação:
Clone este repositório para sua máquina local.
git clone https://github.com/Brunop10/TrabXML.git

Execução:
IntelliJ
1. Abra o IntelliJ.
2. Importe o projeto (File -> Open -> Selecione a pasta do projeto).
3. Certifique-se de configurar o JDK 18.0.2 como o SDK do projeto.
4. Execute a classe Main.java para iniciar o projeto.

VSCode
1. Abra o VSCode.
2. Abra a pasta do projeto.
3. Certifique-se de ter as dependências Java configuradas corretamente.
4. Execute a classe Main.java para iniciar o projeto.
